Build It: Hands-on Projects

Theory is not enough. To truly understand climate technology, you have to build it. Here are implementable projects designed to solidify your skills and build your portfolio. (Covered extensively in Chapter 19).

Beginner

1. Personal Carbon Footprint API & Calculator

Build a web application that takes user inputs (flights, miles driven, diet) and calculates their annual carbon footprint using standard emission factors.

HTML/CSS/JS Node.js EPA Emission Factors
Intermediate

2. Hyperlocal Air Quality Predictor

Train a machine learning model using historical data from OpenAQ combined with weather forecasts to predict the PM2.5 levels for your city 24 hours in advance.

Python Scikit-learn / XGBoost OpenAQ API
Intermediate

3. Satellite Deforestation Detection

Use computer vision and satellite imagery to identify areas of the Amazon that have been cleared over a specific timeframe.

PyTorch Copernicus Sentinel-2 Data CNNs / U-Net
Advanced

4. Smart Building HVAC Reinforcement Learning

Create a simulated environment of a commercial building and train an RL agent to continuously control the HVAC setpoints to minimize energy usage while keeping occupants comfortable.

Python Stable Baselines3 EnergyPlus
Capstone

5. Mini City Digital Twin Dashboard

Build an interactive 3D web dashboard that aggregates open city data (traffic, real-time weather, public transit locations) onto a Mapbox or CesiumJS 3D visualization representing a digital shadow of a city.

Three.js / CesiumJS WebSockets OpenCity Data APIs
